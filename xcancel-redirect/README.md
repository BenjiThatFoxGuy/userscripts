# 🔁 xCancel Redirect: Twitter, But Through xCancel

Sick of Twitter/X? This script auto-redirects every Twitter link to xCancel.com. No login, no tracking, no "please sign in to view". Just tweets, minus the Musk.

---

## Features

- Instantly rewires Twitter/X URLs to xCancel before the page even loads
- Uses xcancel.com as the redirect target
- No setup, no config, just install and forget
